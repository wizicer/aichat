## Deployment

This project is configured for GitHub Pages deployment via GitHub Actions.

### Setup

1. Push your code to GitHub
2. Go to repository Settings â†’ Pages
3. Set Source to "GitHub Actions"
4. Push to `main` branch to trigger deployment

The app will be available at: `https://<username>.github.io/aichat/`

### Configuration

The base path is configured in `vite.config.ts`:
```typescript
base: process.env.NODE_ENV === 'production' ? '/aichat/' : '/'
```

Change `/aichat/` to match your repository name if different.
